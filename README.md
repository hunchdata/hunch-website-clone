# Hunch Website

Marketing website for [hunch.tools](https://hunch.tools)

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Development

```bash
npm install
npm run dev
```

## Deployment

This site is deployed to GitHub Pages via GitHub Actions. Any push to `main` triggers an automatic deployment.
